# 🏷️ Labelit

A modern React application for creating, editing, previewing, and printing shipping and product labels.

**GitHub Repository:**  
https://github.com/anilonall/Labelit

---

# 🇹🇷 Türkçe

## Labelit Nedir?

Labelit, etiket tasarlamayı ve yazdırmayı kolaylaştırmak amacıyla geliştirilen React tabanlı bir web uygulamasıdır.

Uygulama sayesinde;

- Yeni etiket oluşturabilirsiniz.
- Mevcut etiketleri düzenleyebilirsiniz.
- Yazı alanlarını değiştirebilirsiniz.
- Barkod ve QR Kod oluşturabilirsiniz.
- Yazdırmadan önce önizleme yapabilirsiniz.
- Etiketleri PDF olarak kaydedebilirsiniz.
- Etiketleri doğrudan yazıcıya gönderebilirsiniz.

## Özellikler

- ✅ Etiket oluşturma
- ✅ Etiket düzenleme
- ✅ Dinamik alan desteği
- ✅ Barkod oluşturma
- ✅ QR Kod oluşturma
- ✅ PDF çıktısı alma
- ✅ Yazdırma desteği
- ✅ Modern ve responsive arayüz

## Teknolojiler

- React
- JavaScript
- HTML5
- CSS3

## Kurulum

```bash
git clone https://github.com/anilonall/Labelit.git

cd Labelit

npm install

npm start
```

Uygulama varsayılan olarak:

```
http://localhost:3000
```

adresinde çalışacaktır.

# 🇬🇧 English

## What is Labelit?

Labelit is a React-based web application designed to simplify label creation, editing, previewing, and printing.

With Labelit you can:

- Create new labels
- Edit existing labels
- Customize text fields
- Generate Barcodes
- Generate QR Codes
- Preview labels before printing
- Export labels as PDF
- Print labels directly from your browser

## Features

- ✅ Label Creation
- ✅ Label Editing
- ✅ Dynamic Fields
- ✅ Barcode Generator
- ✅ QR Code Generator
- ✅ PDF Export
- ✅ Direct Printing
- ✅ Modern Responsive UI

## Technologies

- React
- JavaScript
- HTML5
- CSS3

## Installation

```bash
git clone https://github.com/anilonall/Labelit.git

cd Labelit

npm install

npm start
```

The application will run at:

```
http://localhost:3000
```
